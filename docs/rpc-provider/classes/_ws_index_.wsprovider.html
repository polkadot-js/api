WsProvider

name: WsProvider

summary: The WebSocket Provider allows sending requests using WebSocket to a WebSocket RPC server TCP port.

description: Unlike the HttpProvider, it does support subscriptions and allows listening to events such as new blocks or balance changes.

example: 

import Rpc from '@polkadot/rpc-core';
import WsProvider from '@polkadot/rpc-provider/ws';

const provider = new WsProvider('ws://127.0.0.1:9944');
const api = new Rpc(provider);
